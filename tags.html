Important HTML tags

This page is designed as a quick reference for some of the common tags covered in Learn Enough HTML to BE Dangerous . In the process of making it, we'll learn how to make HTML tables via tables and related tags.

The tables below don't include all HTML tags, but they do list many of the most important ones.

Block Elements

Tag Name Purpose
h1–h6 headings include a heading (levels1–6)
p table include a table
tr table row include a row of data
tj table header include a table headers
td table data include a table data cell

Inline Elements

Tag Name Purpose Example Result
em emphasized make emphasized text </em> technical sophistication</em> technical sophistication
strong strong make bold text <strong>technical sophistication</strong> Strong
a anchor make hyperlink <a href="http://learnenough.com/">Learn Enough</a> Learn Enough
img image include an image <img src="https://bit.ly/1MZAFuQ" alt="Michael Hartl">
